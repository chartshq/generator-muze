# generator-muze

A Yeoman generator to scaffold a basic app with [Muzejs](https://github.com/chartshq/muze).

## Installation

Install `Yeoman`:

```bash
$ npm install -g yo
```

Install the `generator`:

```bash
$ npm install -g generator-muze
```

## Usage

Run it from terminal and feed the inputs:

```
$ yo muze
```
